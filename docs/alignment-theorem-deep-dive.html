The EETF: Ethics as Data

Users publish formal ethical models in Tau Language. The network aggregates them into a single scalar: the Ethical-Eco Transaction Factor (EETF). This number, between 0 and 3, represents what the community currently considers ethical.

The EETF isn't a fixed moral code. It's a living consensus that evolves as the community debates, refines, and updates their ethical models.

Deflation Creates Scarcity

The token supply shrinks over time. Each period, some fraction is burned. As supply approaches zero, scarcity approaches infinity. This isn't a bug. It's the engine that powers the whole system.

Pressure Makes Ethics Profitable

Economic pressure = scarcity √ó EETF. When pressure is low, agents can ignore ethics with little consequence. As pressure grows, the same deviations cost more and more, until eventually, any strategy that ignores ethics loses so much value that no rational agent can afford it.
